# O NOVO PARADIGMA
O curso que você está fazendo trata do JavaScript Isomorfo, um novo paradigma no desenvolvimento de aplicativos web com frameworks MVC JavaScript. Vou explicar o que isso significa em termos mais simples.

JavaScript Isomorfo, também conhecido como JavaScript Universal ou Server-Side Rendering (SSR), é uma abordagem que permite a execução de código JavaScript tanto no servidor (lado do servidor) quanto no navegador do usuário (lado do cliente). Isso tem se tornado popular porque resolve alguns problemas que surgem ao criar aplicativos da web modernos com frameworks como o Meteor.js.

Aqui estão os principais pontos:

1. **Generalização dos Frameworks MVC JavaScript**: Frameworks como Meteor.js, Angular, React e Vue.js se tornaram populares para desenvolver aplicativos da web. Eles oferecem uma estrutura organizada para o código e facilitam a criação de aplicativos interativos.

2. **Rápida Expansão**: Esses frameworks se expandiram rapidamente devido à eficiência e facilidade que proporcionam no desenvolvimento de aplicativos web, resultando em um grande número de aplicativos sendo construídos com essas tecnologias.

3. **Problemas a Serem Resolvidos**: À medida que os aplicativos web se tornaram mais complexos, surgiram problemas, como baixa otimização para mecanismos de busca (SEO), carregamento lento da primeira página e dificuldades de acessibilidade para mecanismos de busca.

4. **JavaScript Isomorfo**: Para resolver esses problemas, surgiu o conceito de JavaScript Isomorfo. Isso permite que partes do aplicativo sejam renderizadas tanto no servidor quanto no cliente. Dessa forma, você pode melhorar o SEO, reduzir o tempo de carregamento da primeira página e garantir que seu aplicativo seja acessível para mecanismos de busca.
